Full-Stack Blog Application

A full-stack blogging platform built with Express, MongoDB, React, and Vite.
Includes user authentication, protected routes, and complete CRUD functionality for blog posts.

Features:

Backend(Express + MongoDB)

-User registration and login
-JWT authentication
-Protected API routes
-Create, read, update, delete (CRUD) operations for posts
-Authorization checks to ensure only owners can edit/delete
-Input validation and structured error handling

Frontend (React + Vite)

-React Router navigation
-Authentication context and token management
-Protected routes for authenticated users
-Post list and detail views
-Post creation, editing, and deletion
-Form validation and error display
-Responsive interface

Security

-Password hashing (bcrypt)
-JWT token generation and validation
-Authorization based on post ownership
-CORS configuration
-Server-side and client-side validation

Tech Stack

Frontend: React, Vite, React Router
Backend: Node.js, Express, MongoDB, Mongoose
Security: JWT, bcrypt

Summary

This project demonstrates the ability to build a complete, modern full-stack web 
application with authentication, protected APIs, and a responsive user interface. 
It follows current industry standards for both frontend and backend development.
